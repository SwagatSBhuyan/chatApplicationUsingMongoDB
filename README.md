# chatApplication

Simple chat app that uses MongoDB, Node.js and Socket.io

## Pre-requisites:
Node js
Socket.io
MongoDB (Compass and CLI)

## Install Dependencies
npm install

## Run Server
npm start

## Run App
Open index.html
Make two or more instances depending on number of users.
